Manual Técnico
Este manual técnico detalla los pasos necesarios para evaluar la usabilidad, calidad y eficiencia de los sistemas de información en universidades. Contiene métricas de evaluación, ejemplos de implementación, y recomendaciones basadas en estándares internacionales como ISO 9241-11 y ISO/IEC 25010.

Ideal para desarrolladores y administradores de sistemas que buscan mejorar la experiencia de usuario y optimizar el desempeño de sus plataformas.

Guía de Usuario
Esta guía de usuario es un recurso completo diseñado para ayudarte a utilizar la página web creada para evaluar y mejorar los sistemas de información universitarios. Incluye instrucciones detalladas y ejemplos prácticos que abarcan:
  • Cómo usar las herramientas de evaluación disponibles en el sitio.
  • La funcionalidad del cada una de las secciones que contiene la página.
  • El uso del chatbot integrado como asistente para resolver dudas frecuentes.
  • Acceso a recursos como el calendario de fases y la checklist de cumplimiento.
  • Recomendaciones técnicas para desarrolladores interesados en personalizar o ampliar el sistema.

Ideal tanto para usuarios finales como para administradores y desarrolladores que buscan mejorar sus sistemas en términos de usabilidad, navegabilidad y experiencia del usuario.
